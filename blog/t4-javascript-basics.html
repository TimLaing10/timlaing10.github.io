Javascript basics

How does JavaScript compare to HTML and CSS

HTML and CSS provide the layout and design of a webpage. JavaScript is responsible for the interactive elements of the site and tells the browser how to change the web page in response to events that happen.

Explain control flow and loops using an example process from everyday life

Control Flow is a fundamental concept in programming that allows you to dictate how your code runs under different conditions or until a certain condition is met.

An example from everyday life would could be brushing your teeth.

    Do I have my toothbrush?
    Yes (Next step), No, (Teeth stay dirty)
    Do I have toothpaste?
    Yes (Next step), No, (get toothpaste and repeat step)
    Is the tap on?
    Yes (Next step), No, (turn on tap and repeat step)
    Scrub teeth for two mins
    Yes (Kiss the dog goodnight), No, (Keep brushing)
Explain the difference between accessing data from arrays and objects.

JavaScript provides a data type specifically for storing sequences of values called and array and is written between square brackets separated by commas. E.G [4, 5. 6. 8].

An array can be an object, which itself is a collection of properties, and a property is an association between a name (or key) and a value.

Explain what functions are and why they are useful.

Functions are one of the core aspects of a programming language and are the basis to creating useful applications. A function is a saved set of instructions that you want to use many times throughout an application.

An example of an object from the Mozilla foundation follows. var myCar = new Object(); myCar.make = "Ford"; myCar.model = "Mustang"; myCar.year = 1969;
