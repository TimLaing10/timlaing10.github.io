Language

Parentheses () Parentheses, which are (), are used in JavaScript to contain a list of paremeters passed to functions and control structures (the flow of execution of the program). They are also used to group expressions to control the order of execution.

Brackets [ ] Brackets hold the contents of an array, which are used to hold multiple values in a single variable. You refer to an array element by referring to the index number (e.g [0]). An example of this is var meats = ["chicken", "beef", "pork"] where the element [0] would refer to chicken.
